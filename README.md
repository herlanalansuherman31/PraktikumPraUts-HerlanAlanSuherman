## 🔢Penjelasan Folder

### kelas_objek_01
Folder ini berisi contoh dasar penggunaan kelas dan objek dalam bahasa pemrograman Java. Konsep ini merupakan fondasi dari pemrograman berorientasi objek (OOP).

### metode_02
Folder ini menjelaskan konsep metode dalam Java, termasuk penggunaan setter methods untuk mengubah nilai atribut objek.

### parameter_03
Folder ini mendemonstrasikan penggunaan parameter dalam method Java, di mana method dapat menerima nilai sebagai input untuk mengubah perilaku objek.

### Constructor_04
Folder ini menjelaskan konsep konstruktor dalam Java, yang digunakan untuk menginisialisasi objek saat dibuat.

### Enkapsulasi_05
Folder ini menunjukkan prinsip enkapsulasi dalam OOP, di mana atribut objek disembunyikan (private) dan diakses melalui getter dan setter methods.

### Pewarisan_06
Folder ini mengilustrasikan konsep pewarisan dalam Java, di mana kelas anak mewarisi atribut dan method dari kelas induk.

### Polimorfisme_o7
Folder ini menjelaskan polimorfisme dalam Java, di mana objek dari subclass dapat diperlakukan sebagai objek superclass, dan method yang sama dapat memiliki implementasi berbeda.

### SudyCase
Folder ini berisi contoh penerapan konsep OOP dalam skenario nyata, yaitu mengelola data mahasiswa.

## 💻Cara Menjalankan
Setiap folder memiliki file README.md sendiri dengan instruksi detail cara menjalankan program di dalamnya. Secara umum:
1. Masuk ke direktori folder yang diinginkan.
2. Compile dengan: `javac -d . *.java`
3. Jalankan dengan: `java -cp . [nama_package].[nama_kelas_utama]`

Pastikan Java Development Kit (JDK) sudah terinstall di sistem Anda.
